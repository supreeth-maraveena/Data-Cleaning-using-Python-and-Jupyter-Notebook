# 🧹 Data Cleaning using Python and Jupyter Notebook

## 📖 Overview
This project demonstrates a **step-by-step approach to data cleaning using Python in Jupyter Notebook**. It is designed to help understand the process of transforming messy datasets into a clean, analysis-ready format.

The workflow is divided into **10 structured steps**, making it easy to follow and apply to other datasets.

---

## 🛠️ Features
- Uses a real-world dataset: `marketing_campaign_data_messy.csv`  
- Covers **10 structured data cleaning steps**  
- Handles missing values, duplicates, formatting, and outliers  
- Implemented using Jupyter Notebook for interactive analysis  
- Provides a clear and reusable data cleaning methodology  

---

## 🧑‍💻 Tech Stack
- **Language:** Python  
- **Environment:** Jupyter Notebook  
- **Libraries:** pandas, numpy, matplotlib, seaborn

## ▶️ Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/supreeth-maraveena/Data-Cleaning-using-Python-and-Jupyter-Notebook.git
   cd Data-Cleaning-using-Python-and-Jupyter-Notebook
   ```

2. Install required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```

3. Run Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

4. Open `Data_Cleaning.ipynb` and follow the steps:
   - Load the dataset  
   - Perform the 10-step cleaning process  
   - Observe transformations at each stage  
   - Apply the same workflow to other datasets
